## Setup

Install [Git](http://git-scm.com/), [RVM](https://rvm.io/), PostgreSQL, [InfluxDB](https://portal.influxdata.com/downloads#influxdb).

    $ git clone https://github.com/SAkshat/dummyapp.git
    $ cd dummyapp/ # prompted by RVM
    $ bundle
    $ rake db:create
    $ rake db:migrate

Addons Used
===========
Postgres
InfluxDB
